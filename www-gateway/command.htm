Поддерживаемые команды:

  • to (на) - перенаправление на компьютер локальной сети (см. Ответы на вопросы->Ваш компьютер подключен к Интернет через локальную сеть).
  • run (запустить) - запустить файл или папку. Например: run C:\ - запустит папку C:\
  • shutdown (выключить) - выключение компьютера
  • reboot (перезагрузить) - перезагрузить компьютер
  • shut (выйти) - завершение сеанса
  • msg (сообщение) - показывает сообщение. Например: to comp key msg Привет - выведет сообщение Привет на локальном компьютере comp, где запущена программа-сервер с ключом key.
  • define (макроопределение) - работа с макроопределениями. Например: define del "mac" - удаляет макроопределение mac;  define "mac_2" "A B C D F" - добавит/заменит макроопределение mac_2, которое будет заменяться командой (A) с параметром этой команды (B C D F).;define "mac_3" ""A B C D" F" - добавит/заменит макроопределение mac_3, которое будет заменяться командой (A B C D) с параметром этой команды (F). Все аргументы функций берутся в кавычки.
  • stop (остановить) - остановить программу-сервер
  • port (порт) - сменить прослушиваемый порт программно
  • cmd (командная_строка) - поддержка любой команды из командной строки. Например: cmd netsh.exe wlan connect name=HOME - включит беспроводную сеть HOME.
  • выполнения любой команды, которая поддерживается в Пуск->выполнить. Например: regedit c:\1.reg - занесет в реестр данные из файла 1.reg.

 

В скобках указаны команды в русском варианте (равнозначны английскому варианту).
