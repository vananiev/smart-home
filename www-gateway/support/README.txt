Требования к хостингу: PHP 4.0 и выше

ДЛЯ УСТАНОВКИ НЕОБХОДИМО:
1.	Откройте файл index.php через любой текстовый редактор
2.	Найдите 5-ую строку кода: $res = run('mail@example.com'); // <-- Ваш адрес e-mail здесь
3.	Вставьте вместо mail@example.com свой электронный ящик
4.	Далее по коду вы можете изменять тексты, которые отображаются для пользователей
