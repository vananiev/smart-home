Отправить команду с помощью СМС.

Есть вопросы? Задавайте.

Что тебе нужно, чтобы начать работу.

1. Во-первых скачать программу-сервер.

2. При запуске программы придумать ключ  к ней, с помощью которого программа определит, что это именно ты отправляешь ей задания.

3. Ввести логин. Если ты зарегистрирован и решил его указать. При использовании логина облегчится управление программой и отправка команд с нашего сайта.

4. Обновите ваш IP. Для этого войдите в пункт меню Сервис, выберите Настройки соединения и нажмите получить IP. Запомните его и номер порта для отправки команд без входа в систему или, если вы ввели логин, запоминать IP и номер порта не обязательно, информация в вашей учетной записи обновится автоматически.

5. Все готово!

   Запуск программы сервера можно осуществить с параметрами командной строки:  /hide   [key  [login] ] , например /hide 000 sergey - запустит программу скрыто, с ключом 000 и логином sergey;  /hide 000 - запустит программу скрыто, с ключом 000 без логина;  /hide - запустит программу скрыто, с ключом по умолчанию default без логина. Последний пример не рекомендуется использовать.

 

Как отправить команду с сайта.

Есть два способа.

1. Низкоуровневый (для всех пользователей).

На главной странице переходим по ссылке  Отправить команду с помощью IP и номера порта. Вводим IP вашего компьютера и номер порта, указанные в программе (для этого войдите в пункт меню Сервис, выберите Настройки соединения и нажмите получить IP). Вводим команду и ключ к вашей программе. Отправляем. На экране будет выведен результат выполнения на вашем компьютере отосланного запроса.

2. Более удобный (только для зарегистрированных пользователей).

На главной странице переходим по ссылке  Отправить команду с помощью Учетной записи. Если вы еще не вошли в систему, вводим логин и пароль. Указываем команду и ключ к вашей программе. Отправляем. На экране будет выведен результат выполнения на вашем компьютере отосланного запроса.

Внимание! Чтобы ваш компьютер принял запрос на выполнение на нем должна быть запущена программа сервер.

 

Ваш компьютер подключен к Интернет через локальную сеть.

Для правильной работы в сети, нужно:

1. Запустить программу-сервер на вашем компьютере и на компьютере, который непосредственно подключен к Интернет.

2. Указать в ней и в программе-сервере, запущенной на вашем компьютере, одинаковые порты.

При отправке команды с помощью IP и пароля в начале необходимо указывать перенаправление на на ваш компьютер, тогда выполняемая_комманда выглядит следующим образом: to [имя_вашего_компьютера или локальный_адрес] выполняемая_комманда.   

Имя_вашего_компьютера можно посмотреть в Сервис -> Настройки соединения.

Например: to PETR run cmd - запустит командную строку на локальном компьютере с именем PETR.

to 192.168.0.3 run cmd - запустит командную строку на компьютере c локальным адресом 92.168.0.3 .

Перенаправление осуществляется с помощью команды to.

 

В чем преимущество регистрации.

1. Незарегистрированным пользователям возможно отправить команду на свой компьютер, только используя IP и номера порта, которые необходимо держать в голове или переписывать на листочек. Если вы зарегистрировались и указали при старте программы логин, то IP и порт будут обновляться и запоминаться автоматически на нашем сайте.

2. Если ваш компьютер подключен к Интернет через локальную сеть то при отправке команды в начале необходимо указывать перенаправление на на ваш компьютер, тогда выполняемая_комманда выглядит следующим образом: to [имя вашего компьютера или локальный адрес] выполняемая_комманда. Где: Имя компьютера можно посмотреть в меню Сервис -> Настройки соединения. Если вы зарегистрировались и указали при старте программы логин это перенаправление указывать не нужно. 

 

Добавить вашу собственную команду.

Наша программа поддерживает гибкое и в тоже время простое управление командами. Если вы хоть как-то сталкивались с составление программ на любом языке программирования, вы можете легко добавить новую свою собственную команду! Для этого необходимо

1. Написать программу, переименовать исполняемый файл в имя, совпадающее с именем команды, а расширение файла может быть любым.

2. Поместить ее в папку \command, который находится в папке с программой-сервером.

3. Команда готова к использованию!

Ваша программа может принимать параметры командной строки, их необходимо указывать за командой через пробел (если в самой команде или аргументах есть пробел, их необходимо указывать в кавычки. Например: 123 456 интерпретируется как команда 123 с аргументом 456, а "123 456" интерпретируется как команда 123 456.

Также в роли команды может выступать любой другой файл из \system32 (путь к папке указывать не нужно) или любой другой неисполняемый файл (*.com, *.bat и даже *.txt), который помещен в \command.

Важно! Имя программы в папке \command должно быть не повторяющимся, даже если там существует файл с другим расширением, но таким же именем. Иначе одна из команд работать не будет.

 

Использование макроопределений.

Если вам надоедает вводить длинную последовательность символов в команде, то вы можете заменить эту последовательность более короткой  с помощью макроопределения. Макроопределение - это новая команда, которая заменяет другую команду (более длинную, переведенную на другой язык и т.д.). Например: вы часто используете команду A B C D, с помощью макроопределения ее можно заменить командой E, тогда везде, где вы используете E программа автоматически подставит команду A B C D.

Для добавления макроопределения используйте команду define.

Для добавления макроопределения откройте файл word.txt и добавьте Имя и Макроопределение каждую на новой стоке в конце файла (Команды будут находиться на нечетных строках, а Макроопределения - на четных). Имя - заменяемая имя макроопределения, Макроопределение - заменяющая последовательность символов. И помните, что если в какой-то из команд содержится пробел, то необходимо эту команду помещать в кавычки. Например для приведенного выше примера необходимо добавить две строки строку:
E  
 "A B C D" F
Строка понимается как макроопределение Е заменить командой A B C D c параметром F.

Важно! Макроопределение не должно совпадать ни  с какой из существующих команд.

 

Использование функций.

Если необходимо использовать результат выполнения одной команды в параметрах другой команды, необходимо использовать функции.

Рассмотрим пример: msg (define del "mac") - Отобразит в окне  результат функции define del "mac". Как это происходит? Во время выполнения команды msg (define del "mac") все вызовы функций (define del "mac") с символами '(' , ')' заменяются результатом их выполнения. Возможный вариант замены функции: msg (define del "mac") --> msg Макроопределение удалено. Заметьте, что функции всегда записываются в круглые скобки!

Еще один пример:    to   petr   key1   msg   (to  timon  key2  define  "mac"  "A B")...ok    -  отошлет команду msg   (to  timon  key2  define  "mac"  "A B")...ok на компьютер petr, и на компьютере petr выведется сообщение о результате выполнения функции to  timon  key2  define  "mac"  "A B" (на удаленном компьютере timon выполнится команда define  "mac"  "A B"). Возможный результат выведенного сообщения на компьютере petr:  Макроопределение создано...ok. Как видно функцию можно включать и в середине команды как в последнем примере.

Функции записываются в круглые скобки!

 

Отправить команду с помощью СМС.

Отправляете смс на номер 1121 (стоимость услуги)    Есть два способа:

1. Низкоуровневый (для всех пользователей).

                        =send    IP   Порт    [ to локальное_имя/адрес]   Ключ   Команда

где: IP - ip-адрес вашего компьютера, указанный в программе (для этого войдите в пункт меню Сервис, выберите Настройки соединения и нажмите получить IP).

Порт - номер порта, указанный в программе (для этого войдите в пункт меню Сервис, выберите Настройки соединения и нажмите получить IP).

[ to локальное_имя/адрес] - нужно вводить, если Ваш компьютер подключен к Интернет через локальную сеть.  (локальное_имя можно посмотреть в Сервис -> Настройки соединения).

Ключ - ключ указанный при старте программы.

Команда - исполняемая команда.

Например: =send 95.12.13.14 12001 to petr default run cmd. Отправляем. К вам придет СМС с результатом выполнения на вашем компьютере отосланного запроса.

2. Более удобный (только для зарегистрированных пользователей).

                        =send    Логин   Пароль    Ключ   Команда

Логин - ваш логин в нашей системе.

Пароль - ваш пароль в нашей системе.

Заметьте, перенаправление указывать не нужно, системе сделает это за вас, главное вовремя обновляйте информацию при старте программы или смене вашего ip (меню Сервис, выберите Настройки соединения и нажмите получить IP). Например:  =send Серега 12345 run cmd. Отправляем. К вам придет СМС с результатом выполнения на вашем компьютере отосланного запроса.

Внимание! Чтобы ваш компьютер принял запрос на выполнение на нем должна быть запущена программа сервер.
