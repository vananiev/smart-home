Проект по бесплатному удаленному управлению умным домом через СМС.
Суть идеи заключается в том, что СМС, отправленное хозяином дома через СМС шлюз (сторонней компании) доставляется
на www-сервер. Дома установлен сервер, на котором запущено ПО (см smart-home-server/), которое по логину-паролю подключено на www-сервер и периодически получает команды на выполнение.

На текущий момент реализованы тестовые команды (запуска утилит в windows cmd, смены обоев рабочего стола, обращения к выводам параллельного порта через которые могут быть подключены устройста, выполнения сторонних разрешенных бинарных файлов, выключения и перезагрузки сервера).


В директории www-gateway/ находится код, управляющий интернет сервисом, задачей которого являются регистрация нового пользователя и контроль доступа, передача команд от СМС шлюза на зарегистрированную серверную компоненту на стороне Дома.
В директории smart-home-server/ сервер, отвечающий за получение команд из сети, инициированных от смс владельца, и их выполнение.

Код являлся рабочим на состояние декабря 2009 г, однако на текущий момент www-сайт cmd.paradoxos.ru выведен из работы.
